# Dr. Danny Vincent – Psychiatry Home Clinic

Static website for **Dr. Danny Vincent**, a Senior Consultant Psychiatrist based in Cherthala, Kerala.

## Live Site

[drdannyvincent.com](https://drdannyvincent.com/)

## Overview

A single-page informational site for Dr. Danny's Psychiatry Home Clinic, featuring:

- Doctor profile and professional background
- Services offered (anxiety, depression, addiction psychiatry, child & adolescent mental health, etc.)
- Clinic hours and contact details
- Appointment booking via phone, WhatsApp, and email
- Embedded Google Maps location

## Tech Stack

- HTML, CSS, vanilla JavaScript
- No frameworks or build tools
- Google Analytics (gtag.js)
- Structured data (Schema.org JSON-LD) for SEO

## Project Structure

```
├── index.html          # Main page
├── css/styles.css      # Stylesheet
├── js/year.js          # Dynamic copyright year
├── images/             # Doctor photos (responsive srcset)
├── sitemap.xml         # Sitemap for search engines
├── robots.txt          # Crawler directives
├── site.webmanifest    # PWA manifest
└── _headers            # Custom HTTP headers
```
